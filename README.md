## 基础知识

+ 荷兰拍
    * 优点
        - 雷电网络荷兰拍卖时适逢熊市，募集的eth不多，发币后上了币安有好几倍的收益
        - 如果募集的eth多的话, 收益也就没多少了
            + 对于任何项目来说，估值过高都会影响到投资人收益

   * 筹的代币数目或者代币金额达到确定的上限时立刻结束，硬顶可以认为是ICO达到其最高融资额度就停止的指标。
